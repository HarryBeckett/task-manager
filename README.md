# task-manager
A simple task manager written in C using arguments instead of command line. While I'm well aware that this might not be the most efficient way to carry out this project it was definitley a good learning experience for me.


#How to use
The usage of this program is very simple. It uses only arguments from the command line, there are 3 commands; "add", "remove", and "list". for any exra help needed add help as you r arguemnt for a help list or check out the code.
